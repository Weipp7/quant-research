## 第零章：为什么是多因子？

### 多因子 vs 网格的本质区别

你已经做透了网格策略，知道它的天花板在哪：网格本身不创造alpha，它依赖标的的"震荡 + 重心上移"。

**多因子选股是完全不同的范式**：

| 维度 | 网格策略 | 多因子选股 |
|---|---|---|
| 核心问题 | "怎么交易" | "买什么" |
| Alpha来源 | 震荡 + 标的趋势 | 选股能力 |
| 持仓风格 | 中长期持有，逢低加仓 | 定期轮动（月度/周度） |
| 标的选择 | 主观选少数几只 | 系统性筛选数十只 |
| 理论根基 | 行为金融学（均值回归） | 资产定价理论（CAPM/APT） |
| 收益预期 | 年化6-8% | 年化10-15% |
| 回撤特征 | 跟随标的回撤 | 通常更分散，但极端行情仍有大回撤 |

**为什么多因子能突破10%**：
- 它解决的是"选股"这个核心问题
- 通过组合多个因子的"小概率优势"，叠加成"高胜率优势"
- 历史上头部量化私募的核心策略基本都是多因子的某种变体

## 第一章：理论基础（必读）

### 1.1 资产定价的演进史

理解多因子，先理解它从哪里来。

**第一阶段：CAPM（1964）**

夏普提出资本资产定价模型，核心思想：**所有风险资产的预期收益由唯一因子决定——市场组合的收益**。

公式：`E(R_i) = R_f + β_i × (E(R_m) - R_f)`

含义：股票的预期超额收益 = β × 市场超额收益。也就是说，股票越激进（β越大），就该获得越多回报。

**问题**：实证上，CAPM解释力有限。很多股票的超额收益无法用β解释——比如小市值股票长期跑赢大市值，价值股长期跑赢成长股。这些"异象"CAPM搞不定。

**第二阶段：APT（1976）**

罗斯提出套利定价理论，核心思想：**收益不只由一个因子决定，可以由多个因子共同决定**。

但APT没明确指出"哪些因子有效"——这是个开放性框架。

**第三阶段：Fama-French三因子模型（1993）**

法玛和弗伦奇用实证给出了答案：除了市场因子，**市值因子（SMB）**和**账面市值比因子（HML）**也能解释横截面收益差异。

公式：`E(R_i) - R_f = β_m × MKT + β_s × SMB + β_h × HML`

- **MKT（市场因子）**：市场组合超额收益
- **SMB（Small Minus Big）**：小市值组合 - 大市值组合的收益差。捕捉"小盘股长期跑赢大盘股"的现象
- **HML（High Minus Low）**：高账面市值比组合 - 低账面市值比组合的收益差。捕捉"价值股跑赢成长股"的现象

法玛因这个模型2013年获得诺贝尔经济学奖，**这是现代多因子投资的理论基石**。

**第四阶段：Carhart四因子（1997）**

加入**动量因子（MOM）**：过去12个月涨得多的股票，未来一段时间继续涨。这个加进去后模型解释力又显著提升。

**第五阶段：Fama-French五因子（2015）**

法玛和弗伦奇又加了两个：
- **RMW（Robust Minus Weak）**：盈利能力强 - 盈利能力弱的收益差
- **CMA（Conservative Minus Aggressive）**：投资保守 - 投资激进的收益差

到这一步，"什么因子有效"的科学版图基本成型。

### 1.2 为什么这些因子有效？

理解每个因子背后的**经济逻辑**比记住公式重要：

**市值因子 SMB（小盘溢价）**
- 小公司风险高（流动性差、抗冲击能力弱）→ 投资者要求更高回报
- 注意：A股的小盘溢价历史上极强，但2017年后弱化（注册制、退市加速、监管趋严）

**价值因子 HML（账面市值比效应）**
- 行为金融解释：投资者过度悲观，把价值股压得太低 → 均值回归带来超额收益
- 风险解释：价值股往往是"困境股"，承担额外风险

**动量因子 MOM（趋势延续）**
- 行为金融解释：投资者反应不足（信息扩散需要时间）+ 羊群效应
- A股特点：**短期动量（1月）反而是反转的**，中期动量（3-12月）才有效

**盈利因子 RMW（质量溢价）**
- 高ROE公司有"护城河"，长期更有可能创造超额价值
- 巴菲特式投资的量化版本

**投资因子 CMA（保守投资偏好）**
- 过度扩张的公司往往未来表现不佳（管理层过度自信）
- 资本支出增速快的公司，其未来回报反而低

### 1.3 因子模型的两种用法

**用法一：风险归因**（学术用途为主）

把组合收益拆分成"哪些因子贡献了多少"。比如基金经理的alpha到底是真本事还是只是暴露在某个因子上。

**用法二：选股**（实战用途）

如果你相信"小市值+低估值+高ROE+动量"的股票未来跑赢，那就**主动选这些股票**构建组合。这就是多因子选股策略的核心。

我们要学的是**用法二**。

## 第二章：多因子选股的完整流程

业界普遍把多因子选股拆成5步，这是固定的方法论：

```
1. 因子选取 → 2. 单因子有效性检验 → 3. 因子筛选与去冗余 
→ 4. 多因子合成与打分 → 5. 组合构建与持续监控
```

下面逐一拆解。

### 2.1 因子选取（找候选）

A股常用因子分**6大类**：

#### 估值因子（Valuation）
衡量股票"贵不贵"
- **PE（市盈率）** = 价格 / 每股收益
- **PB（市净率）** = 价格 / 每股净资产
- **PS（市销率）** = 价格 / 每股营收
- **PCF（市现率）** = 价格 / 每股现金流
- **EP（盈利收益率）** = 1/PE，**A股最常用，比PE更稳定（避免负值问题）**
- **BP（账面市值比）** = 1/PB

**经验**：A股的BP/EP在中长期（季度以上）有效，PE在短期效果不稳定。

#### 成长因子（Growth）
衡量公司"成长快不快"
- **营业收入同比增长率**
- **净利润同比增长率**
- **ROE增长率**
- **每股收益（EPS）同比增长率**

**经验**：成长因子在A股单独使用效果一般，但与质量因子结合（PEG型）效果较好。

#### 质量因子（Quality）
衡量公司"基本面好不好"
- **ROE（净资产收益率）**
- **ROA（总资产收益率）**
- **毛利率**
- **资产负债率**（反向）
- **经营性现金流/营业收入**

**经验**：质量因子在A股长期有效，与价值因子结合（巴菲特式）特别强。

#### 动量因子（Momentum）
衡量股票"近期涨势如何"
- **过去N月收益率**（N通常取1、3、6、12）
- **52周新高距离**
- **指数加权动量**（最近的收益权重更高）

**经验**：A股的动量因子有特殊性——
- 1个月动量是**反转**的（短期反转效应）
- 3-12月动量是**延续**的
- 所以构建动量因子时，常用 `return_12m - return_1m`（去掉最近1个月）

#### 波动率因子（Volatility）
衡量股票"涨跌剧烈程度"
- **过去N日收益率标准差**
- **特异波动率**（剔除市场因素后的波动率）
- **β值**（与市场相关性）

**经验**：低波动因子在A股长期有效，**低波动股票比高波动股票收益更高**——这是反直觉的"低波动异象"。

#### 流动性因子（Liquidity）
衡量股票"交易活跃度"
- **换手率**（当日成交量/流通股本）
- **Amihud流动性指标**

**经验**：低流动性因子在A股有溢价（流动性差的股票需要补偿），但在大资金体量下不能用（自己就把流动性吃掉了）。

#### 规模因子（Size）
- **总市值**
- **流通市值**

**经验**：A股小市值因子在2017年前是"圣杯"，2017年后弱化但仍有效。

### 2.2 单因子有效性检验

光知道这些因子还不够，**必须验证它们在你的目标市场和时间段是否真的有效**。这是最核心的工作。

业界三种主流方法：

#### 方法一：IC分析（最常用）

**IC（Information Coefficient，信息系数）**：因子值与未来收益的相关系数。

具体计算：
1. 在每个调仓时点（比如每月初），计算所有股票的因子值
2. 等下一个调仓周期结束，计算每只股票实际收益率
3. 对这两组数据求相关系数 → 这就是这一期的IC

**IC的两种算法**：
- **Pearson IC**：普通线性相关。对异常值敏感
- **Rank IC（推荐）**：先把因子值和收益率分别转成排名，再求相关。**更稳健，业界主流用Rank IC**

**判断标准（Rank IC均值的绝对值）**：
- |IC| < 0.02：因子基本无效
- 0.02 ≤ |IC| < 0.05：弱有效
- 0.05 ≤ |IC| < 0.10：有效
- |IC| ≥ 0.10：强有效（罕见，要警惕过拟合）

**IR（Information Ratio）= IC均值 / IC标准差**

衡量因子稳定性。IR > 0.5 就算不错的因子。

#### 方法二：分层回测（最直观）

把股票按因子值排序，分成5-10层（业界叫"分位数"），等权持有每层一段时间，看哪一层收益最高。

**判断标准**：
- **单调性**：从低分位到高分位，收益应该单调递增（或递减）
- **多空收益**：多头组合（最高分位）- 空头组合（最低分位）的收益。这个差越大，因子越有效
- **稳定性**：在不同年份多空收益是否稳定为正

**直观例子**：测试"低PE效应"，把全A股按PE分5组：
- 第1组（PE最低）年化15%
- 第2组 12%
- 第3组 10%  
- 第4组 8%
- 第5组（PE最高）年化5%

完美的单调性，多空收益10%。这就是个好因子。

#### 方法三：回归法

横截面上跑回归：`收益率 = α + β × 因子值 + 残差`

看β的t统计量是否显著。这个方法更严谨但也更复杂，适合学术研究，**实战中IC + 分层回测就够用**。

### 2.3 因子预处理（关键步骤）

测因子前，**必须先做数据清洗**。否则得到的IC、收益率都不可靠。

标准流程：**去极值 → 标准化 → 中性化**

#### 去极值
原始因子常常有极端值（比如某只ST股票的PE是10000）。这些值会污染回归和相关性计算。

主流方法：
- **MAD法（中位数绝对偏差法）**：以中位数±N倍MAD为边界，超出的截断
- **3σ法**：以均值±3σ为边界
- **百分位法**：取1%-99%分位区间，外面截断

**推荐MAD法**——对正态性假设要求最低。

```python
def mad_winsorize(factor_series, n=3):
    """MAD去极值"""
    median = factor_series.median()
    mad = (factor_series - median).abs().median()
    upper = median + n * 1.4826 * mad  # 1.4826是正态分布修正系数
    lower = median - n * 1.4826 * mad
    return factor_series.clip(lower, upper)
```

#### 标准化
把不同量纲的因子转到统一尺度，方便横向对比和合成。

**Z-score标准化**：`新值 = (原值 - 均值) / 标准差`

```python
def zscore(factor_series):
    return (factor_series - factor_series.mean()) / factor_series.std()
```

#### 中性化（重要但常被忽略）

**问题**：很多因子之间有内在相关性。比如PB和市值——大市值公司往往有更稳定的盈利和更高的PB。如果不处理，**你以为在选低PB，实际上是在选小市值**。

**解决方案：用回归剔除其他因子的影响**

```python
# 例：对PB做市值中性化
import statsmodels.api as sm

X = sm.add_constant(market_cap_log)  # 市值对数
y = pb_factor                         # PB因子
model = sm.OLS(y, X).fit()
neutralized_pb = model.resid          # 残差就是中性化后的PB
```

**业界常做的中性化**：
1. **市值中性化**：剔除规模影响
2. **行业中性化**：剔除行业影响（不同行业天然PE/PB差距大）

中性化做完后，因子才是"纯净"的，IC才能准确反映因子本身的预测力。

### 2.4 多因子合成

通过单因子检验后，挑出几个**有效**且**互相相关性低**的因子，把它们合成一个综合得分。

#### 第一步：剔除冗余因子

计算因子之间的相关系数。如果两个因子相关性>0.7，**留一个就行**。

经典组合（低相关性）：
- 估值（BP）+ 动量（12-1月）+ 质量（ROE）+ 波动（低波动）+ 规模（小市值）

#### 第二步：因子打分

对每个有效因子，按因子值排序打分。比如：
- 因子值最高的1%股票 → 100分
- 因子值最低的1%股票 → 0分
- 中间线性映射

#### 第三步：因子加权合成

合成方法（按复杂度递增）：

**等权法**：每个因子权重相同。最简单，鲁棒性最好。
```
综合分 = (BP分 + ROE分 + 动量分 + 低波动分) / 4
```

**IC加权法**：用历史IC的均值作为权重
```
权重_i = IC_i / Σ IC_j
综合分 = Σ (权重_i × 因子分_i)
```

**IC_IR加权法**（业界主流）：用IC_IR作为权重，给"稳定有效"的因子更大权重
```
权重_i = IC_IR_i / Σ IC_IR_j
```

**最大化IC_IR法**：通过协方差矩阵优化权重。复杂但理论最优。

**初学者建议从等权法开始**——简单粗暴但效果常常不输复杂方法。

### 2.5 组合构建

#### 选股
按综合得分排序，选**Top N**只股票。
- N通常取30-100只
- N太少：组合方差大，受单只股票影响大
- N太多：稀释alpha

#### 等权 vs 加权
- **等权**：每只股票投入相同金额。简单稳健
- **市值加权**：大股票多投。容易被龙头股拖累
- **得分加权**：得分高的多投。alpha更集中但风险更高

**推荐等权**。

#### 调仓周期
- **月度调仓**（推荐）：alpha衰减不明显，交易成本可控
- **周度调仓**：交易成本高，需要更强的因子
- **季度调仓**：alpha衰减明显，但极适合基本面因子

### 2.6 风险控制

实战中必须加的风险控制：

1. **行业分散**：单个行业持股不超过组合的30%
2. **个股仓位上限**：单只股票不超过5%
3. **流动性筛选**：剔除日均成交额<5000万的股票
4. **过滤特殊状态**：剔除ST、停牌、上市不足90天的股票
5. **大盘择时（可选）**：大盘极弱时降仓位

### 3.2 第一个最简单的多因子策略：5因子等权打分

策略思路：
- 股票池：沪深300成分股（流动性好，避免坑）
- 因子：BP（估值） + ROE（质量） + 动量12-1（动量） + 1/波动率（低波） + 1/市值（小盘）
- 合成：等权打分
- 选股：Top 30
- 持仓：等权
- 调仓：月度

**伪代码框架**：

```python
def initialize(context):
    set_benchmark('000300.XSHG')
    g.universe = '000300.XSHG'  # 股票池：沪深300
    g.top_n = 30                # 选30只
    
    # 月度调仓
    run_monthly(rebalance, monthday=1, time='open')

def get_factor_data(context, stock_list):
    """获取因子原始数据"""
    # 1. 估值：BP（账面市值比）
    bp = get_factor_values(stock_list, ['book_to_price_ratio'], 
                            end_date=context.previous_date, count=1)['book_to_price_ratio'].iloc[-1]
    
    # 2. 质量：ROE_ttm
    roe = get_factor_values(stock_list, ['roe_ttm'], 
                             end_date=context.previous_date, count=1)['roe_ttm'].iloc[-1]
    
    # 3. 动量：过去12月-1月
    prices_12m = history(252, '1d', 'close', stock_list)
    prices_1m = history(20, '1d', 'close', stock_list)
    momentum = (prices_12m.iloc[-21] / prices_12m.iloc[0]) - 1  # 12月前 vs 21天前
    
    # 4. 波动率：过去20日std
    returns = prices_1m.pct_change().std()
    low_vol = -returns  # 取负，让低波动得高分
    
    # 5. 市值
    fundamentals = get_fundamentals(query(
        valuation.code, valuation.market_cap
    ).filter(valuation.code.in_(stock_list)), date=context.previous_date)
    market_cap = fundamentals.set_index('code')['market_cap']
    small_cap = -market_cap  # 取负，让小市值得高分
    
    return pd.DataFrame({
        'BP': bp, 'ROE': roe, 'MOM': momentum, 
        'LowVol': low_vol, 'SmallCap': small_cap
    })

def winsorize_and_standardize(factor_df):
    """去极值 + 标准化"""
    result = factor_df.copy()
    for col in result.columns:
        # MAD去极值
        median = result[col].median()
        mad = (result[col] - median).abs().median() * 1.4826
        result[col] = result[col].clip(median - 3*mad, median + 3*mad)
        # Z-score标准化
        result[col] = (result[col] - result[col].mean()) / result[col].std()
    return result

def rebalance(context):
    # 1. 获取股票池
    stock_list = get_index_stocks(g.universe, context.previous_date)
    # 过滤ST、停牌、新股
    stock_list = filter_stocks(stock_list)
    
    # 2. 获取因子数据
    factor_df = get_factor_data(context, stock_list)
    factor_df = factor_df.dropna()
    
    # 3. 去极值 + 标准化
    factor_df = winsorize_and_standardize(factor_df)
    
    # 4. 等权合成
    factor_df['score'] = factor_df.mean(axis=1)
    
    # 5. 选Top 30
    target_stocks = factor_df.sort_values('score', ascending=False).head(g.top_n).index.tolist()
    
    # 6. 调仓
    rebalance_to_target(context, target_stocks)

def rebalance_to_target(context, target_stocks):
    """调仓到目标持仓（等权）"""
    # 卖出不在目标里的
    for stock in context.portfolio.positions:
        if stock not in target_stocks:
            order_target(stock, 0)
    
    # 买入目标股票（等权）
    target_value_per_stock = context.portfolio.total_value / len(target_stocks)
    for stock in target_stocks:
        order_target_value(stock, target_value_per_stock)

def filter_stocks(stock_list):
    """过滤ST/停牌/新股"""
    current_data = get_current_data()
    filtered = []
    for stock in stock_list:
        if current_data[stock].is_st: continue
        if current_data[stock].paused: continue
        # 上市超过250天
        info = get_security_info(stock)
        if info is None: continue
        if (datetime.date.today() - info.start_date).days < 250: continue
        filtered.append(stock)
    return filtered
```

这个策略麻雀虽小五脏俱全。你可以先把这个跑起来，看看效果。我估计沪深300内年化能做到10-13%。

### 3.3 进阶方向

跑通第一版后，可以逐步加入这些进阶要素：

**A. 单因子检验体系**
- 用jqfactor_analyzer分别测每个因子的IC、IR、分层
- 找出真正有效的因子，剔除无效的
- 这一步是把"猜"变成"知道"

**B. 因子中性化**
- 加入市值中性化、行业中性化
- 避免因子之间隐性重叠

**C. 因子筛选**

**D. IC加权合成**

**E. 风险控制**

## 第四章：避坑指南

### 4.1 必须避开的雷区

**1. 未来函数（Look-ahead bias）**

最致命的错误。常见形式：
- 在月初决策时使用了月底的财务数据（财报有滞后）
- 用调整后的复权数据但没考虑当时还没调整
- 用了"当前已知"的成分股列表（实际是后来调入的）

**修复**：
- 财务数据用`statDate`参数指定真实可获得的截止日
- 用`use_real_price=True`并理解复权机制
- 沪深300成分股用对应日期的成分（`get_index_stocks(date=...)`）

**2. 幸存者偏差**

只测试现存股票，没考虑退市股。比如选小市值因子，2015年的小盘股很多后来退市了，但你的回测里它们都"活到了今天"，导致小市值因子被高估。

**修复**：聚宽默认提供已退市数据，确保你的股票池包含历史所有股票。

**3. 过拟合**

因子参数（如动量的回看期）通过反复调试得到一个看起来很好的回测，但实盘失效。

**修复**：
- 参数尽量用业界默认值
- 训练集 + 测试集分割，最后用测试集验证
- 用滚动窗口做样本外测试

**4. 交易成本未充分考虑**

A股交易成本不低（佣金万3 + 印花税千1卖出 + 滑点）。月度调仓换手率100%，年化成本就是0.04% × 12 × 2 = 1%。如果是周度调仓，年化成本能吃掉5%以上。

**修复**：回测设置真实成本，关注换手率，避免不必要的调仓。

**5. 流动性陷阱**

回测说选了100只股票每只投1万。但有些小市值股票日均成交才500万，你1万下去，下一秒成交价就跑了。

**修复**：
- 加流动性过滤（日均成交额>5000万）
- 单股仓位 / 日均成交额 < 1%

### 4.2 思维上的坑

**误区1：因子越多越好**

错。因子之间相关性高的话，加入只是噪音。**3-5个低相关、高IC的因子比10个高相关因子好**。

**误区2：IC越高越好**

警惕。IC太高（>0.15）往往是过拟合或未来函数的信号。健康因子的IC在0.05-0.10之间。

**误区3：在A股直接照搬美股因子**

A股有自己的特点：
- 散户主导（行为偏差更强）
- T+1限制
- 涨跌停
- 政策驱动明显
- 风格切换更快

很多美股因子在A股要做调整或者根本无效。

**误区4：忽略市场环境变化**

2017年之前小市值因子是A股圣杯，年化20%+。2017年之后小市值因子持续不灵。**因子是会失效的，要持续监控**。
