# 聚宽（JoinQuant）策略归档

Phase 1 在聚宽云端跑出的策略快照。

**所有策略代码使用聚宽专有 API**（`from jqdata import *`、`run_daily`、`order_value` 等），**只能在聚宽 IDE 里运行**，不能直接在本地执行。这里保存的是历史快照，用于 Phase 3 本地复现时的对照基准。

## 共同回测条件

- 标的：平安银行（000001.XSHE）单股
- 基准：沪深 300（000300.XSHG）
- 区间：2019-12-30 至 2024-12-31（5 年）
- 手续费：佣金万三 + 卖出印花税千一，最低 5 元
- 复权：动态复权（真实价格）

## 每个策略目录的结构

```
<strategy>/
├── strategy.py           聚宽 IDE 里的源码（只在聚宽上能运行）
├── backtest_report.png   回测报告截图
├── daily_summary.csv     按日聚合的回测数据（已从 GBK 转 UTF-8）
├── metrics.md            从截图提取的全部指标（数字版，便于对比 / Phase 3 对照）
└── observations.md       "我观察到的 3 件事"（待你填写）
```
